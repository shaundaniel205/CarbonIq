# CarbonIQ

Live Demo: https://carbon-iq-peach.vercel.app/

CarbonIQ is a web application that helps users track their daily carbon footprint based on transportation, food habits, and energy consumption.

The goal of the project is to make users more aware of how their daily activities affect the environment and encourage sustainable habits.

## Features

* User signup and login
* Carbon footprint tracking
* Activity logging
* Dashboard with emissions overview
* Profile management
* Transportation, food, and energy tracking

## Tech Stack

* Next.js
* React
* TypeScript
* Tailwind CSS
* Supabase
* Vercel
